# Структура работы

**Подготовка данных**
Загрузка данных: Использован датасет Palmer Penguins с информацией о характеристиках трех видов пингвинов.

Обработка пропущенных значений: Удалены строки с пропущенными значениями

Кодирование категориальных признаков:

Бинарные признаки (Clutch Completion, Sex) закодированы через map()

Многоклассовые признаки (Species, Island) закодированы с помощью LabelEncoder

Целевая переменная Species закодирована ординально: Chinstrap=0, Gentoo=1, Adelie=2

**Разделение данных**
Данные разделены на тренировочную и тестовую выборки в соотношении 70/30

Использована стратификация для сохранения распределения классов

Зафиксирован random_state=42 для воспроизводимости результатов

**Обучение моделей kNN**
Обучено 6 классификаторов kNN с разным количеством соседей: 1, 3, 5, 10, 15, 25

Использованы только два признака: Flipper Length и Body Mass

Признаки масштабированы с помощью StandardScaler

Использовано евклидово расстояние

**Визуализация результатов**
Установлена и использована библиотека mlxtend для визуализации решающих поверхностей

Построена сетка графиков (2×3) для всех шести классификаторов

На графиках отображены границы принятия решений и тренировочные данные

Ключевые результаты
Точность моделей:

Визуальные наблюдения
Меньшие значения k создают более сложные границы решений

Большие значения k дают более гладкие границы

Оптимальный баланс достигается при k=25
